Cloud Computing

Tentang

Pelajari dasar-dasar komputasi awan untuk menyebarkan aplikasi, memantau operasi, dan mengelola solusi perusahaan dengan spesialisasi berikut:

  • Dukungan TI Google
  • Yayasan Komputasi Awan
  • Jalur Pembelajaran Rekayasa Awan
  • Web Dasar
  • Javascript Dasar
  • Dasar Back-End
  • Insinyur Awan
  • Pengantar Kecerdasan Buatan
  • Pembelajaran Mesin Terapan untuk Insinyur Cloud
  • Pengembangan Aplikasi dengan Cloud Run

Prasyarat

Pengetahuan dasar tentang tumpukan jaringan IP, basis data, statistik, dan ANSI SQL.

Miliki laptop yang menjalankan spesifikasi minimum berikut:

  • RAM minimal 2GB
  • Chrome OS, Windows 7 atau versi lebih baru, macOS X, atau Linux
  • Prosesor 32-bit atau 64-bit
  • Peramban web Google Chrome
  • Menginstal editor teks profesional (VS Code, NotePad++, Sublime Text, Atom, dll.)
